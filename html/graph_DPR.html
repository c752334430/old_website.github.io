Dot Product Represent for graph(summary)

Mainly summarize the idea and history of graph representation, especially DPR.

Posted by Tianyi Chen on Nov 22, 2018

Given a graph G = (V,E), we can store it as:
1. Adjacency list representation (store neighbors).
2. Adjacency matrix (most common).
3. Dot product representation (v_i * v_j = 1 if connected and 0 if disconnected for every i and j in graph).

Some definitions:

Given a graph G on vertices V = {v1,...,vn} and a field F, an asymptotic dot product representation of G, of dimension d over F, is a sequence of mappings {Xk} where each Xk : V → Fd such that for i ̸= j, lim(Xk(vi)·Xk(vj)) equals 1 if vi ∼ vj, and equals 0 otherwise.

For a field F and a graph G, dpF(G) [dpF∗ (G)] is the minimum dimension d such that Fd admits an exact [asymptotic] dot product representation of G.(clearly we want to achieve a small d)

The matrix X is an exact dot product representation of G iff XT X = A + D, where A is the adjacency matrix of G and D is some diagonal matrix.

Let Pn be the path on n vertices. For any field F, dpF(Pn) = n−1.

For any field F and graph G on n vertices, dpF(G) ≤ n − 1, where n is the number of nodes. And dpF(G) ≤ m, where m is the number of edges

Problem for today's web dataset.
